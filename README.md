## Toegangkelijkheid

De toegangkelijkheid van de website is met verschillende manieren getest voor optimale gebruik voor mensen met een beperking, zodat iedereen de website kan blijven gebruiken. Alle klikbare elementen blijven altijd toegangkelijk voor iedereen, en staan ze op logische volgorde

## Huisstijl
